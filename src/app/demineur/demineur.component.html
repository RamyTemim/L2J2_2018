Démineur

Règles :

Voici ci-dessus une grille de démineur contenant un certain nombre de mines à vous de la résoudre sans vous faire exploser !!!!


Pour mieux comprendre comment jouer voici un résumer des règles du démineur

Lorsque vous cliquez sur une case trois situations sont possible :

  • 1ère situation : la case est vide -> il n'y a donc pas de mines sur l'une des huit cases qui l'entourent, celles-ci seront dévoilées et si l'une d'entre elle est vide il se passera la même chose et ainsi de suite jusqu'à tomber sur une case contenant un nombre (elle sera dévoilée) ou une mine (elle restera cachée).
  • 2ème situation : la case contient un nombre -> celui-ci indique le nombre de mines situées sur les huits cases adjacentes, la case est alors dévoilée et uniquement celle-ci.
  • 3ème situation : la case contient une mine -> vous avez perdu, toutes les mines de la grille sont dévoilées.


  • Information importante :
    • Il se peut qu'après un clic sur une case vide, toutes les cases vide indirectement adjacentes à celle cliquée ne soient pas dévoilées...
    • Pas de soucis car dans ce cas vous savez que toutes les cases adjacentes à une case vide ne contiennent pas de mines
    • Vous ne prendrez donc aucun risque en cliquant dessus :)
    • Mais ATTENTION uniquement celles qui touchent directement la case vide dévoilée ne comportent pas de risque (i.e celles en : haut/bas; digonale; droite/gauche)

Pour résoudre nos grilles de jeu vous devrez faire bien attention au contenu de chaque case pour deviner petit à petit où sont cacher toutes les mines

Commandes :

  • Clique sur une case :dévoile les éléments adjacents à la case
  • Clique droit une première fois : permet de placer un drapeau sur une case
  • Clique droit une deuxième fois : permet de retirer le drapeau de la case
  • Pour lancer le chrono lors de la première partie vous devez cliquer sur "JOUER", ensuite il se lancera automatiquement à chaque début de partie et le bouton "JOUER" disparaîtra
  • Bouton "valider" -> affiche un message indiquant si vous avez trouver toutes les mines (et stop le chrono) : pour cela un drapeau doit être placé sur chacune d'entre elles
  • Bouton "solution" -> affiche la solution de la grille et stop le chrono
  • Bouton "nouvelle grille" -> affiche une nouvelle grille jouable (il est possible de choisir un niveau différent avant de cliquer dessus)

Bonne chance !!!!
